## About

Quick launcher and search for files, apps, and web.

## Features

- App launching
- File search
- Web search
- Calculator
- Quick Look

## Installation

### From App Store
Open the **App Store** in zOS and search for "Spotlight".

### Via npm
```bash
npm install @zos-apps/spotlight
```

## Usage

- Type to search
- Arrow keys to navigate
- Enter to open
